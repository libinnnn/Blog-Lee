# Blog-Lee
个人博客搭建


1、clone代码到地  git clone https://github.com/libinnnn/Blog-Lee.git

2、在IDEA中以maven的方式进行导入，同时安装lombok插件

3、导入数据库，根目录下的blog.sql文件，同时在application-dev.yml中修改好对应的数据库地址，以及用户名密码

4、运行BlogApplication
